API Reference

Base URL (local): http://localhost:5000

Overview Quickstart API Reference

Notification Object

This is the shape of a notification returned by the API:

{
  "id":            "abc123",
  "title":         "Welcome!",
  "message":       "Thanks for using the app.",
  "targetScreen":  "home",
  "audience":      "all",
  "isActive":      true,
  "showOnce":      false,
  "startDate":     "2024-01-01",
  "endDate":       "2099-12-31",
  "viewsCounter":  5,
  "clicksCounter": 2
}

Endpoints

GET /api/notifications

Returns active notifications filtered by screen and user type. Used by the Android SDK.

Query ParamDescription
screen optionalFilter by targetScreen value (e.g. "home")
userType optionalFilter by audience (e.g. "regular", "premium", "all")
// Response
{
  "notifications": [ { ...notification }, ... ]
}
GET /api/admin/notifications

Returns all notifications including inactive ones. Used by the admin portal.

{
  "notifications": [ { ...notification }, ... ]
}
GET /api/notifications/:id

Returns a single notification by ID.

// 200 OK
{ ...notification }

// 404 Not Found
{ "error": "Notification not found" }
POST /api/notifications

Creates a new notification.

Body FieldTypeDescription
titlestringrequiredNotification title
messagestringrequiredNotification body text
targetScreenstringrequiredScreen name (e.g. "home")
audiencestringrequired"all", "regular", or "premium"
startDatestringrequiredYYYY-MM-DD format
endDatestringrequiredYYYY-MM-DD format
showOncebooleanoptionalDefault: false
// 201 Created
{ ...notification with id }
PUT /api/notifications/:id

Updates an existing notification. Send the full notification object in the body.

// 200 OK
{ ...updated notification }

// 404 Not Found
{ "error": "Notification not found" }
PATCH /api/notifications/:id/activate

Sets isActive to true.

// 200 OK
{ ...notification with isActive: true }
PATCH /api/notifications/:id/deactivate

Sets isActive to false.

// 200 OK
{ ...notification with isActive: false }
PATCH /api/notifications/:id/view

Increments viewsCounter by 1. Called automatically by the SDK when a notification is shown.

// 200 OK
{ "success": true }
PATCH /api/notifications/:id/click

Increments clicksCounter by 1. Called automatically by the SDK when the user taps "Learn More".

// 200 OK
{ "success": true }
DELETE /api/notifications/:id

Deletes a notification permanently.

// 200 OK
{ "success": true }

// 404 Not Found
{ "error": "Notification not found" }
